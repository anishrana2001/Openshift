# Red Hat EX188 / DO188 Podman Practice Labs

Welcome to the Podman practice lab series.

These labs are designed for students learning Red Hat containers, Podman commands, image builds, volumes, networks, pods, secrets, troubleshooting, Kubernetes YAML generation, and systemd integration.

## How to Use

1. Start from Task 01.
2. Complete each task in sequence.
3. Run the validation commands.
4. Answer the student questions.
5. Perform cleanup only after verification.

## Suggested Student Workflow

```bash
git clone <your-repo-url>
cd <your-repo-name>
```

Then open each Markdown file and complete the steps.
