# 📌 OpenShift Core Concepts

## 🎯 Objective

Understand the building blocks of OpenShift.

---

## 🧠 Key Concepts

### 🔹 Cluster

A set of nodes running containerized apps.

### 🔹 Node Types

* Master Node
* Worker Node

### 🔹 Pod

Smallest deployable unit in OpenShift.

---

## 📦 Example

```bash
oc get nodes
oc get pods -A
```

---

## ⚠️ Common Mistakes

* Confusing pod with container
* Ignoring namespaces

---

## 🔥 Exam Tip

👉 Always check resources using:

```bash
oc get all
```
